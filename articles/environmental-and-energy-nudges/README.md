# Environmental and Energy Nudges

Article folder for **Environmental and Energy Nudges** in the Choice Architecture and Nudging knowledge series.

## Folder layers

- calculators: self-contained calculator scripts and smoke tests
- data: synthetic or documented data files
- docs: article-specific documentation
- notebooks: exploratory notebooks
- outputs: generated outputs
- python, r, julia, sql, haskell, rust, go, c, cpp, fortran, java, typescript, prolog: language-specific examples

## Notes

Use synthetic data unless real public data is explicitly documented.
